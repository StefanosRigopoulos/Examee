Welcome to Examee

The procedure is simple

1. Get the Exam Renderer

In order to start configuring your own examination
you need our build tool. You can find the library
with everything you will ever need through the link
below. Simply import the DLL file into your project.

ExameeRenderer.dll

2. Create the Exam

The tool is very versatile and can help you
create a very unique set of exam. You can
create as many question as you want and also
be able to provide your parameterson how you
want to structure your examination.

Documentation

3. Generate the Exams

When you are done configuring the exam, head
over to the upload page to use our generator
to get your exams with the parameters you have
provided.
